# Contributing to apps

Thank you for your interest in this repository!

## Read-Only Access

This repository is **publicly viewable but read-only** for external users. 

- You can view all code and files
- You can fork the repository for your own use
- You can open issues to report bugs or suggest improvements

## Write Access

Write access to this repository is restricted to:
- Repository owner: @ToddButch71
- Approved collaborators only

## Questions or Suggestions?

If you have questions or suggestions, please feel free to open an issue for discussion.
